ServiceNow Hackathon

Overview

Designed and developed a COVID test scheduler application using ServiceNow Platform, with 3 teammates in ServiceNow + Accenture Hackathon held at Northeastern University. The application allows users to schedule RT-PCR Tests in university test centers. Email notifications notifies user about their upcoming tests based on testing cadence, results, overdue COVID test notification. The application has a dashboard designed to indicate stats on testing like total number of tests, number of COVID positive cases etc., in real time.
