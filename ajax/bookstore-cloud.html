Online Bookstore

Project Overview

Online Bookstore is a web application that is configured to be hosted on Amazon Web Services (AWS) cloud computing platform. The sole purpose of this project was to explore AWS platform and implement best DevOps practices. The entire Infrastructure was built as a Code (IaaC) using Terraform.

Project Details

  • Application Type:Web Application
  • Language:Java
  • Framework:Spring MVC, Hibernate
  • Cloud Computing Platform:AWS
  • Database:AWS RDS - MySql
  • AWS Resources: VPC, RDS, S3, ELB, Auto Scaling, Route53, AWS Lambda, DynamoDB, SNS, SES, CodeDeploy, Cloud Watch agent
  • Infrastructure as Code:Terraform
  • CI/CD Pipeline:CircleCI
  • URL:Online Bookstore
