Camp World

Project Overview

Camp world is web application built for people who love camping! The application allows campers across the world to add campgrounds that they have been to and share pictures. Also, campers can write reviews on campgrounds, share their experience and rate it. The application is built using Node.js and Express.js framework and hosted on Heroku cloud platform.

Project Details

  • Application Type:Web Application
  • Languages:JavaScript, Node.js, HTML, CSS
  • Frontend Framework:Bootstrap 5
  • Backend Framework:Express.js
  • Database:MongoDB
  • Cloud Platforms:Heroku, MongoDB Atlas, Cloudinary
  • URL:Camp World
